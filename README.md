# Places Attribute Conflation
Project A: Winter 2026

Creating a single reliable record from mutliple location sources.


Sample data:
The file contains pre-matched pairs of place records. Each row represents a pair, where the prefix "base_" marks one place record and the other does not have a prefix. The following columns are 
present: ['id', 'base_id', 'sources', 'names', 'categories', 'confidence',
       'websites', 'socials', 'emails', 'phones', 'brand', 'addresses',
       'base_sources', 'base_names', 'base_categories', 'base_confidence',
       'base_websites', 'base_socials', 'base_emails', 'base_phones',
       'base_brand', 'base_addresses'].

Further information on the schema per place can be found here: https://docs.overturemaps.org/schema/reference/places/place/
